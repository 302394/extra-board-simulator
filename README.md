# Extra Board Simulator Web App v5

Run:

```bash
npm install
npm run dev
```

Adds:
- Tappable hold-down day buttons for week vacations that do not start Monday
- Actual Tie-Up Adjustments labels hold-down jobs as HOLD-DOWN for the regular
- Starting Board Order at the top of every day
- Ending Board Order keeps resting/unavailable people on the board with availability notes
- Separate Relief and Vacation sections


## v6 note

Hold-down days are selected with day-of-week buttons. Leave all unchecked to use the regular's normal work days.


## v7 note

Rest is now calculated from the return leg start time to return tie-up, not from the original outbound call time.
